Detailed Description

template<typename Graph, typename Length, typename Vertex = vertex_of_t<Graph>>
class arlib::details::SkylineContainer< Graph, Length, Vertex >

A conventient container for labels dominance checking.

When a new label l' for a node n is created we must check it against Lemma 2: Let P_LO the set of computed alternative paths so far. If all the labels already existing for node n have a similarity with P_LO that is less than the similarity of l' with P_LO, then l' is dominated and should be pruned.
