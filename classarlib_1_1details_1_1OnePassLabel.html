Detailed Description

template<typename Graph, typename Length, typename Edge = edge_of_t<Graph>, typename Vertex = vertex_of_t<Graph>>
class arlib::details::OnePassLabel< Graph, Length, Edge, Vertex >

A label for a node in the graph to keep track of its exploration state.

A label tracks the path from the source to the node n it's attached, the similarity of the path p(s -> n) wrt to the alternative paths computed so far and the time step the similarities were updated (i.e. the number of alternative paths against which the similarities are currently computed)

Labels can either be head labels if they are attached to source node, or have a predecessor label, i.e. they are attached to a node n such that there exist a label attached to a node n' and an edge (n', n) in Graph.
