Sorts elements in [first, last) by minimum relative similarity, that is:

  • The first element is the path with minimum length
  • Let s be the number of sorted elements
  • Until k elements are sorted, find first element in [first+s, last) such that its similarity with all the already-sorted elements is less than theta. If you don't find, pick the one with minimum similarity.
Precondition
Elements in [first, last) are sorted by length
