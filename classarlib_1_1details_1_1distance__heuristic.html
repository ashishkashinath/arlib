Detailed Description

template<typename Graph, typename CostType>
class arlib::details::distance_heuristic< Graph, CostType >

An A* heuristic using distance from target lower bound.

In order to derive tight h(n, t) lower bounds, we first reverse the edges of the road network and then run Dijkstra’s algorithm from target t to every node n of the network

Template Parameters
GraphA Boost::PropertyGraph having at least one edge property with tag boost::edge_weight_t.
CostTypeThe value type of an edge weight of Graph.
