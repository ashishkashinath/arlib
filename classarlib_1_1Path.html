Detailed Description

template<typename Graph, typename Vertex = vertex_of_t<Graph>, typename Edge = edge_of_t<Graph>, typename Length = length_of_t<Graph>>
class arlib::Path< Graph, Vertex, Edge, Length >

A view of a graph to a simple path.

A (simple) Path from s to t is a cycle-free connected sequence of edges \(p_{st} = \langle (s, v_1), (v_1, v_x), \dots , (v_y, t) \rangle\).

This class wraps a boost::filtered_graph<Graph> to expose only vertices and edges that make up a simple path. Moreover it provides the length of the path, which is defined as the sum of the weights of all the edges. In formula: \(\ell(p) = \sum_{\forall (u, v) \in p} w_{uv}\).

Just like boost::filtered_graph, Path models the same concepts as the underlying graph type. If the underlying Graph type models VertexAndEdgeListGraph and PropertyGraph then so does the Path. If the underlying Graph type models fewer or smaller concepts than these, then so does the Path.
